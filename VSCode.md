# Connect to GitHub with VS2022

## Prerequisites
Before you begin, ensure you have the following:
- Visual Studio Code 2022 installed on your computer. 
- A GitHub account.

## Steps to Connect to GitHub

1. **Install the GitHub Extension for VS2022**
    - Download VS2022.

2. **Sign in to GitHub**
    - Opening vs2022 and sign in your Github account.

3. **Open the Repository**
    - Click Porject1.sln to start working on your project.

4. **Make Changes and Commit**
    - Go to the Source Control view in VS2022.
    - Stage your changes, write a commit message, and commit the changes.

5. **Push Changes to GitHub**
    - After committing, click the `Sync Changes` button in the Source Control view.
    - This will push your changes to the GitHub repository.


By following these steps, you can easily connect and manage your GitHub repositories using Visual Studio Code 2022.
